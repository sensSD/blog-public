# 分治法

将问题拆分成多个互相独立的子问题，递归求解

## 题目

![](/blogs/Test3/2696d345f1ddff51.png)

分块就肯定是分治思想。快排的时间复杂度很熟悉了，最好情况就是O(nlogn)，最坏的情况是把所有的元素遍历一遍然后再对每个分块进行排序，复杂度O(n^2)。

> 答案：分治，D

## 递归

- S(n) = O(n)

# 回溯法

## 深度优先算法

列出所有选择枝，对每一种可能进行计算，当当前枝不能达成最优时，回溯上一步重新选择。

## 题目

![](/blogs/Test3/6891b63413acfd6a.png)

0-1背包问题，要么全都要，要么全都不要。求最大装包，那就把最大价值的物品放入背包，算出来最大值为15。时间复杂度就相当于把c数组遍历一遍，为nW（i为物品数量n，j为背包容量W）

> 答案：C，A

![](/blogs/Test3/f76761d435999b63.png)

当为完全背包问题时，背包装满才结束，对此，我们需要算出每个物品的性价比（v/w），分别为3，1.5，0.83，0.8，1.5。选出最大的几项，当没装满，但背包有空间时，用``` 当前最大性价比 x 剩余背包空间 ```，得出最大价值为16.67。因为完全背包采用了归并排序，所以时间复杂度可以优化到O(nlgn)

> 答案：D，B
