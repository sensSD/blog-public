# 查找
### 二分查找
- 算法核心：mid = (low + hight) / 2
- 时间复杂度：O(logn)
### 分块查找
- 特点：块内无序、块间有序。
- 第一步在索引表中确定待查记录所在快，第二部块内顺序查找
## 哈希表查找
- 特点：数据元素的关键字与其存储地址直接相关

# 排序
## 插入排序
- 将待排序的元素插入到已排序好的子序列中，直到全部完成
## 希尔排序
- 通过增量d对各子表进行插入排序，直至d = 1
## 冒泡排序
- 两两比较，直到排序完成
## 快速排序
- 在待排序表中取一个元素p，以它为基准左右分治
## 简单选择排序
- 在每一趟中选出最大（最小）的元素加入有序子序列
