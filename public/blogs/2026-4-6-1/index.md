# TCP/IP协议族

- TCP/IP分层模型<br>
	从上而下分为**应用层、传输层、网际层和网络接口层**

- 网络接口层协议
- 网际层协议 一 IP
- ARP和RARP<br>
	地址解析协议(ARP)及 反地址解析协议(RARP)。ARP的作用是将IP地址转换为物理地址，RARP的作用是将物理地址转换为IP地址。
- 网际层协议 一 ICMP
- 传输层协议 一 TCP<br>
	传输控制协议(TCP)为应用程序提供了一个可靠的、面向连接的数据传输服务。
- 传输层协议 一 UDP<br>
	用户数据报协议(UDP)是一种不可靠的、无连接的协议，可以保证应用程序进程间的通信。<br>
	TCP有助于提供可靠性，而UDP则有助于提高传输的高速率性。

# IP地址和IPv6简介

- 域名<br>
	一个完整、通用的主机域名由一下四部分组成<br>
	计算机主机名.本地名.组名.最高层域名

- IP地址<br>
	主机地址的唯一标识

## IP地址

IP 地址由**两部分组成，一部分为网络地址，另一部分为主机地址。**

IP 地址可分为5类
- A类：地址范围是1.0.0.1～126.255.255.254。
- B类：地址范围是128.0.0.0～191.255.255.254。
- C类：地址范围是 192.0.1.1~223.255.255.254。
- D类：地址范围是224.0.0.1～239.255.255.254。用在多点广播(Multicast)中
- E类：以1111开始，为将来使用保留，仅作实验和开发用

## IPv6

用于替代IPv4的下一代IP协议。

- 地址长度为128位，空间增大了2^96倍
- 灵活的IP报头格式。加快了报文处理速度
- 简化了报头格式，字段只有8个，加快转发，提高了吞吐量
- 提高安全性
- 支持更多服务类型
- 允许继续演变

# Internet服务

- DNS域名服务：采用UDP端口，**端口号是53**
- 远程登录服务：Telnet用的是TCP端口，**端口号一般是23**
- 电子邮件服务：有邮件传送协议SMTP和用于接收邮件的POP3协议，均采用TCP端口。**SMTP端口号是25，POP3端口号是110**
- WWW服务：交互式图形界面Internet服务。用的TCP端口，**端口号80**
- 文件传输服务：用于在计算机间传输文件。在客户机与服务器内部建立两条TCP连接，一条用于**传输命令和参数（端口号21）**，另一条是**数据连接，用于传送文件（端口号20）**
